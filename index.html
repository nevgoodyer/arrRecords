"Arrays of Records" in 4 languages.

Visual Basic (VB), JavaScript (JS), Python (Py) and Pascal (Pas) all have data structures for "arrays of records". This repository uses "student" records with string, integer and float data types as the example.

The repository for this site is at: https://github.com/bwattle/arrRecords

The site gives practical examples of the NSW Software Development & Design syllabus, sections 8.2.2 & 9.2.2. 
After an overview of the syntax for each language, students will use Visual Basic and possibly JavaScript to build a Pizza ordering app, consolidating concepts of: 
  • adding additional fields and the corresponding screen elements.
  • updating data and screen element dictionaries to keep track of additions.
  • ergonomics, consisting of "appropriate" screen elements and "tab order".
    • text boxes
    • radio buttons
    • check boxes
    • a data picker
    • dropdown boxes for times
  • constantly updating version numbers in:
    • form header (VB & html)
    • page title (html)
    • links below
  • validation, so that the app is "unbreakable" in 2 ways:
    • on submit.
    • on exit from each text box.
    The 1.05 version is easily broken with an invalid date or character instead of a number.
  • the creation of a case insensitive search, possibly including non "utf-8" characters.
  • constantly dealing with VS "Errors", "Warnings" and "Messages", so that code is high quality.
  • handling "infinite loop" (runtime) and "logic" errors.
  • using the student database as a template for the development of a similar "Pizza" ordering database
